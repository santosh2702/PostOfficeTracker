# Post Office Nearby - Find Nearest Delivery Post Office in India

A simple, powerful, and **mobile-friendly** web app that helps millions of Indians find the **nearest Delivery Post Office** using their current location or pincode. Built with real India Post data (165k+ post offices).

### Why This App?
- **E-commerce users** struggle to know where their parcels actually go.
- **Rural users** often don't know the nearest **Delivery office** (not just any BO).
- **Government schemes** (pensions, savings accounts, etc.) need accurate post office info.
- **No other app** reliably shows only **Delivery post offices** with distance & map.

This app solves a real need for 1.4 billion Indians.

## Features
- Detect your current location automatically (works on mobile too)
- Search by pincode or village name
- Shows **top 5 nearest Delivery post offices**
- Displays distance, name, pincode, district & state
- Interactive map with your location and nearby offices
- Clean, India-specific data (invalid coordinates removed)
- Supports English and Hindi
